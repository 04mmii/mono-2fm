# Exhibition Homepage (Vite + React)

Minimal scaffold demonstrating an exhibition-style homepage layout with an oversized turntable on the right.

Run locally:

```bash
npm install
npm run dev
```

Notes:
- Left column constrained to 560px.
- Background `#E4D3CC`.
- Serif typography for titles via Google Fonts.
- Record rotates only when you toggle `Play` (text link).
- Minimal, text-only navigation.
